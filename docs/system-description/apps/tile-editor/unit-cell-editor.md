# "Unit Cell Editor"

"Unit Cell Editor" is the interactive canvas for creating and manipulating polygon instances.

Back to ["Tile Editor"](./index.md).

## User interactions
- Double-click empty space to create a new polygon instance with a new type.
- Double-click an existing polygon to clone it along a ray from its center to the click point.
- Click to select a polygon instance for editing.
- Drag a polygon to move it; drag the background to pan.
- Scroll to zoom.
