# "Current Values"

"Current Values" is the read-only panel that summarizes view settings and selected polygon measurements.

Back to ["Tile Editor"](./index.md).

## Contents
- View offset and constants summary.
- Selected polygon position, side count, and angle sum.
- Per-edge side lengths and per-vertex angles (radians and degrees).

## Behavior
- Updates on selection changes and when expressions are re-evaluated.
- Shows error values in red when expressions are invalid or polygons are open.
