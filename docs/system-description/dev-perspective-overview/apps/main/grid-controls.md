# "Grid Controls"

Back to ["Main"](./index.md).

The "Grid Controls" panel is a Tweakpane UI that organizes configuration into folders. It exposes settings for grid geometry, rendering, particle behavior, and file operations.

Folders and notable controls:
- File (collapsed by default): Save to PNG, Load from PNG.
- Grid (expanded by default): Draw State selector, Grid Scale, Grid Type, Grid Width, Grid Height.
- Particles (expanded by default): Particle Speed, Edge Selection Rule, Orbit Distance (shown only for Orbit Cursor rule).
- GUI (collapsed by default): Palette, Edge Width, Edge Color.
- Advanced (collapsed by default): Left Click Mode, Number of States, Show Coordinates, Visualize Edge Delta, Edge Palette.
