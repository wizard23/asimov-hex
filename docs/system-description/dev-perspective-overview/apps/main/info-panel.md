# "Info Panel"

Back to ["Main"](./index.md).

The "Info Panel" is a fixed overlay panel created in `src/apps/main/index.ts`. It provides high-level context for the app and navigation links.

Features:
- Tabs: About and Tech Stack.
- About tab: description text plus links to Tile Editor, Project Timeline, and Project Statistics.
- Tech Stack tab: static list of libraries and versions.
- Minimize/restore controls to collapse the panel while keeping a small reopen icon visible.
