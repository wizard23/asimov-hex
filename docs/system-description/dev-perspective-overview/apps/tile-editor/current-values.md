# "Current Values"

"Current Values" is a read-only view of the current editor state.

Back to ["Tile Editor"](./index.md).

## Contents
- View offset.
- Constants and their evaluated values (or errors).
- Polygon info for the selected instance, including position, sides, angle sum, and divergence if not closed.
- Per-edge side lengths and per-vertex angles (shown in radians and degrees).
