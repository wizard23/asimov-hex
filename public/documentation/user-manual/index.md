# User Manual

This manual covers all available apps in this project. Each app has its own page with detailed usage instructions.

- [Main App](./main/index.md): interactive grid painter with particle simulation controls.
- [Tile Editor](./tile-editor/index.md): polygon tiling editor driven by expressions and constants.
- [Project Timeline](./timeline/index.md): interactive viewer for the git commit timeline.
- [Project Statistics](./statistics/index.md): explorer for generated repository statistics.
